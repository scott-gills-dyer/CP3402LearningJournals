## Content Insights
Child themes helped me understand that WordPress sites should not be modified directly at the core theme level, as changes can be overwritten when the theme updates. This demonstrated how theme inheritance allows developers to isolate their changes in a separate layer, allowing for better consistency and customisation without affecting the main theme files.

## Career / Employability / Learning Insights
Understanding how to extend or customise software without altering its core components is directly relevant to roles in IT support and deployment.
Gaining experience with these practices in a local Docker development setup also reinforces my experience in testing and development in isolated environments, which will be helpful in future careers.
