## Content Insights
This week I learned how Joomla works as a CMS and how to create and manage content through its administrator interface. The CloudAccess tutorials were useful as they showed the full setup process, which made it easier to follow along. Compared to building a site with HTML and CSS, using templates made it much faster to get a working website without focusing heavily on layout and styling. However, I found Joomla’s interface less intuitive, due to naming pages articals and puting options to edit templates behind system settings, which made some basic tasks feel more complicated than expected.  

## Career / Employability / Learning Insights
This week showed me why CMS platforms like Joomla are used in industry, as they allow content to be updated quickly without needing to modify code directly. This is especially useful in environments where non-developers need to manage websites. I also ran into issues where tutorials were based on older versions, which highlighted how important it is to adapt to changing systems. Being able to read documentation and adjust to different versions is a practical skill that would be expected in real-world IT or web roles.
