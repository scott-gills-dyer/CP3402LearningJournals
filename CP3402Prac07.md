## Content Insights  
While the concepts were familiar from other languages, applying them in PHP required more attention to syntax and structure. 
I found that small mistakes could break the script, so I had to be more precise and test things as I went. 
It also helped me understand how PHP can be used to control output dynamically rather than just displaying static content.  

## Career / Employability / Learning Insights  
This week showed me the importance of being able to write and understand backend logic, even at a basic level. 
In real-world roles, this kind of logic is used to control how data is processed and displayed, not just how things look. 
It also reinforced that developers regularly rely on documentation to confirm syntax rather than memorising everything.
